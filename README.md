# Business Agent – StudyMate AI

This project implements a smart chatbot agent for a fictional business called **StudyMate AI**.

## Features
- Answers questions about the business using TXT and PDF files
- Collects customer leads (name, email, notes)
- Records unanswered questions as feedback using tool functions
- Runs as an interactive chatbot using Gradio

## Files
- `business_agent.ipynb` – Main notebook with agent logic
- `me/business_summary.txt` – Business summary
- `me/about_business.pdf` – Detailed business description
- `leads.jsonl` – Stored customer leads
- `feedback.jsonl` – Stored customer feedback
